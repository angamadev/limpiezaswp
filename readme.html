Instalación: Famosa instalación en 5 minutos

  1. Descomprime el paquete en un directorio vacío y sube todo el contenido.
  2. Abre wp-admin/install.php en tu navegador. Te guiará para crear el archivo wp-config.php con los datos de conexión a tu base de datos.
    1. Si por alguna razón esto no funciona, no te preocupes. Puede que no funcione en todos los servidores. Abre wp-config-sample.php con un editor de texto como WordPad o similar y completa tus datos de conexión.
    2. Guarda el archivo como wp-config.php y súbelo.
    3. Abre de nuevo wp-admin/install.php en tu navegador.
  3. Una vez creado el archivo de configuración, el instalador creará las tablas necesarias para tu sitio. Si hay un error, revisa tu archivo wp-config.php y vuelve a intentarlo. Si sigue fallando, por favor visita los foros de soporte de WordPress.
  4. Si no introdujiste una contraseña, anota la que se te proporcione. Si no introdujiste un nombre de usuario, será admin.
  5. Luego el instalador te llevará a la página de acceso. Ingresa con el usuario y contraseña que elegiste. Si se generó una contraseña para ti, podrás cambiarla en tu perfil.

Actualización

Usando el Actualizador Automático

  1. Abre wp-admin/update-core.php en tu navegador y sigue las instrucciones.
  2. ¿Esperabas más? ¡Eso es todo!

Actualización Manual

  1. Antes de actualizar, asegúrate de tener copias de seguridad de todos los archivos que hayas modificado, como index.php.
  2. Elimina tus antiguos archivos de WordPress, conservando los que hayas modificado.
  3. Sube los nuevos archivos.
  4. Accede a /wp-admin/upgrade.php desde tu navegador.

Migrar desde otros sistemas

WordPress puede importar contenido de varios sistemas. Primero debes instalar y configurar WordPress como se describió anteriormente, y luego usar nuestras herramientas de importación.

Requisitos del Sistema

  • PHP versión 7.2.24 o superior.
  • MySQL versión 5.5.5 o superior.

Recomendaciones

  • PHP versión 7.4 o superior.
  • MySQL versión 8.0 o superior o MariaDB versión 10.5 o superior.
  • Módulo mod_rewrite de Apache.
  • Soporte para HTTPS.
  • Un enlace a wordpress.org en tu sitio.

Notas Finales

  • Si tienes sugerencias, ideas o encontraste algún error, participa en los Foros de Soporte.
